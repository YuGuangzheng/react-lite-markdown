React Lite Markdown
===================

React markdown renderer of [Talk](http://jianliao.com)

Usage
-----

lib

```coffeescript
LiteMarkdown = require 'react-lite-markdown'
```

style (optional, you can @import your own prefer.)
```less
@import "~react-lite-markdown/style/markdown.css";
```
